S02.ex3
-------

Deux versions équivalentes sont disponibles, à choix : 

- WhatComplexitySwing.jar
- WhatComplexity.jar (nécessite une installation de JavaFX)

Le plus immédiat est probablement ceci, depuis IntelliJ IDEA :

  Right-click on WhatComplexitySwing.jar
  -> Run 'WhatComplexitySwing.jar'

[ Hors de l'IDE, le lancement du programme dépend un peu de votre configuration 
d'installation : peut-être simplement par un double-clic sur le fichier 
téléchargé; ou sinon ouvrir une fenêtre de commande (Command Prompt) :
  C:\Program Files\Java\jdk15\bin> java -jar C:\Temp\WhatComplexitySwing.jar
Il faut alors que l'exécutable 'java.exe' soit reconnu : 
- en donnant son chemin complet
- en se déplaçant dans le répertoire (maybe C:\Program Files\Java\jdk15\bin)
- en ajustant la variable d'environnement PATH ]
